Cactus Project Details

Project Summary

This project is a fully automated plant care solution designed around the ESP32-CAM microcontroller. It features:

  • Capacitive soil moisture sensors for accurate hydration monitoring
  • A servo motor that rotates the plant to ensure even exposure
  • A water pump that activates when moisture drops below a set threshold
  • A custom 3D-printed enclosure, designed in SolidWorks and printed using the ELEGOO Neptune 3 Pro
  • An integrated LCD display showing real-time moisture, angle, and elapsed time
  • A responsive web interface (local and remote) built with HTML, CSS, and JavaScript, for manual control and status monitoring

Circuit description

  • This circuit uses an ESP32 as its brains to monitor soil moisture, rotate the plant, and dispense water as needed.
  • The whole system is powered by a 5V, 2Amp wall adapter.
  • Two capacitive soil moisture probes feed analog signals into the ESP32’s ADC pins, and a small MOSFET switches the 5 V water pump on and off.
  • A micro-servo connected to another GPIO rotates the plant container for even exposure.
  • Real-time data moisture level, rotation angle, and elapsed time since last watering is shown on a standard 16×2 LCD hooked up via six data pins.
  • A potentiometer provides contrast control for the LCD, and a few pull-down resistors and decoupling capacitors keep everything stable.
